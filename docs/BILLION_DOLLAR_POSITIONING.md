## PRINCIPLE: Build for Agents First, Humans Second

The dashboard exists for oversight and approval. The actual operators are AI agents. Every feature must be:
- Executable by an agent with API-level access
- Observable by a human in the dashboard
- Approvable by a human with one-click
- Auditable in the system with full provenance

This is the moat. Competitors build dashboards for humans to click through. We build a system that agents operate and humans supervise.
