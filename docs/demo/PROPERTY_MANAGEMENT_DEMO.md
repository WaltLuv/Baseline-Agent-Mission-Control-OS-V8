# Property Management — 15-Minute Demo Script

**Audience:** a property manager / PM company owner. **Goal:** they understand the value in **under 5 minutes**, see proof by minute 10, understand pricing by 15.
**Narrative:** "Your AI property-operations team — installed in minutes, supervised, with receipts for every action."

## Cold open (0:00–2:00) — the problem, then the org chart
- "You're drowning in maintenance tickets, vendor coordination, owner approvals, and leasing follow-up."
- Log in → **Install the Property Management Workforce** (`/app/activate?template=property-management`).
- In seconds the **Org Chart** generates: Tenant Relations, Maintenance Dispatcher, Vendor Coordinator, Owner Relations, Inspections. *"That's your AI staff — already org'd."*

## Value in 5 (2:00–5:00) — run the maintenance flow
- Trigger **Maintenance Intake → Vendor Dispatch → Owner Approval** (`pm-maintenance`).
- Watch it live: tenant request triaged → property/lease matched → vendor checked → dispatch drafted → **owner approval gate before spend**.
- *"Nothing happens behind your back — spend waits for the owner's tap."*

## Proof, not promises (5:00–9:00) — Agent Activity + Replay + Proof
- Open **Agent Activity**: the agents, tools, files, approvals — live, no ghosts.
- Open **Workforce Replay** (`/app/replay`): replay the maintenance mission like a screen recording — trigger → agents → tools → approval → proof package.
- Show the **Proof package**: work order + vendor ETA + owner-approval request + follow-up. *"Every action has a receipt."*

## Supervisor layer (9:00–12:00) — Flight Deck + Hermes + Maestro
- **Flight Deck**: runtime health, your workforce status — the control tower.
- **Maestro**: type a goal ("dispatch a vendor for unit 4B leak") → it routes lane/workforce/provider/approval/cost.
- **Hermes Operator**: tool/skill/provider/runtime registries — what your AI staff can actually do.

## Scale + trust (12:00–14:00)
- **Org Chart** scales: add leasing, vendor, owner-relations pods.
- **Multi-tenant + roles**: invite your team (`/api/workspaces/invites`), assign roles, every action workspace-scoped.

## Close + pricing (14:00–15:00)
- **Billing**: transparent credit ledger, 2.5× markup shown, autoreload — *"you see exactly what each AI action costs."*
- CTA: "Install your workforce today; approvals keep you in control; replay + proof keep you covered."

## Demo prerequisites (set up before the call)
- Seeded admin account; PM template installed once to warm the workspace.
- A connected runtime (or run the supervised simulation path — proof/replay still real).
- Comms provider connected if showing live tenant messaging (else show the drafted-message + approval gate).
