## 2. Sales sheet

Closings slip on one thing: a missing doc, an unflagged appraisal
exception, or a borrower who didn't know they had a deadline.

Baseline deploys four AI employees:

- **AI Doc Collection** chases every missing piece. Adaptive cadence
  (text first, then email, then call), state-specific compliance,
  borrower-friendly copy. The doc cycle drops from 7 days to 36
  hours in pilot data.
- **AI Loan Coordinator** keeps the pipeline current. Borrower
  knows the next step. Lender knows the next deadline. You know
  what's at risk.
- **AI Rate Quote Bot** takes pre-qual inquiries 24/7, pulls from
  the live rate sheet, and books a follow-up with the LO.
- **AI Underwriting Watch** reads the file as it builds and flags
  exceptions before they hit the UW desk — appraisal low, DTI
  drift, missing reserves.

Mission Control supervises. Every borrower communication waits for
the LO's approval. Every memory call cites the loan file or the
state regulation.

**Trial:** $1 / 14 days. **Tier:** $299/mo. GLBA + state regs in
scope from day one.

## 4. Objection handling

| Objection | Response | Proof |
|-----------|----------|-------|
| "Compliance — AI can't talk to borrowers." | "AI drafts; LO approves. State-specific copy reviewed before send. Audit trail per touch. Cleaner than a junior LOA texting unmonitored." | Show approval queue + audit log |
| "Encompass / Calyx already does pipeline." | "Pipeline tracking ≠ pipeline acceleration. AI Loan Coordinator chases the gaps; Encompass shows the gaps. Different jobs." | Demo the chase flow |
| "Borrowers don't text." | "Pilot data: 74% prefer SMS. Adaptive cadence falls back to email + call if SMS isn't responsive." | Show channel waterfall |
| "Rate quotes are LO work." | "Quotes are LO sign-off. Drafts are $20/hr work. AI drafts in 30 seconds; LO approves." | Quote drafting flow |
| "Our compliance officer will block this." | "Bring them in early. Audit log + memory citations + approval gates are exactly what they want to see. We've shipped under GLBA + state regs in the pilot cohort." | Send security doc |
| "We tried a chatbot — borrowers hated it." | "Chatbot ≠ AI workforce. AI Doc Collection is a 4-touch sequence with human handoff. Not a 'connecting you to an agent' loop." | Demo |
