## 2. Sales sheet

AI agencies have a delivery problem masquerading as a sales problem.
Every client wants their own AI workforce. Every client gets a
hand-rolled stack. Five clients in, you're a delivery team, not an
agency.

Baseline gives agencies the operating system:

- **Per-client workspaces** with isolated memory, billing, and
  approvals. One client never sees another.
- **Pre-built vertical templates** (PM, GC, Home Services, Real
  Estate, Mortgage, CPA, Law Firm, Marketing Agency) drop in.
  Customize the SOP, not the stack.
- **Mission Control Supervisor** orchestrates across clients.
  Weekly client reports drafted automatically. Per-client P&L.
- **Demo share links** mint instantly. A salesperson on a call
  hands over a watermarked, signed demo with the prospect's name
  in 10 seconds.
- **Runtime supervision** for Hermes, OpenClaw, Claude Code.
  Install once at the agency level, redeploy per client.

Mission Control supervises. Baseline OS coordinates. Hermes /
OpenClaw / Claude Code execute. Your agency runs production AI
operations, not custom builds.

**Trial:** $1 / 14 days with one test client. **Tier:** $799/mo
Workspace for up to 25 client workspaces.

## 3. Discovery script

1. How many active client engagements is the agency running?
2. What's your average client onboarding time today?
3. Are you hand-rolling client stacks or using a platform?
4. Per-client billing / approvals / memory — separated or shared?
5. What's your average client MRR?
6. How are sales handoffs going from your sales rep to delivery?
7. Weekly client reporting — manual or automated?
8. Last client churn — root cause?
9. Are you supervising AI runtimes already (Hermes / OpenClaw / Claude Code)?
10. $1, 14-day pilot on one test client, full agency workspace?

## 4. Objection handling

| Objection | Response | Proof |
|-----------|----------|-------|
| "We built our own platform." | "Great. Use it for differentiated work. Use ours for the 80% that's the same across clients. We do per-client memory, billing, approvals, and runtime supervision — focus your engineers on the 20% that wins clients." | Demo workspace isolation |
| "Clients want bespoke." | "Vertical templates are starting points, not endings. Configure the SOP per client. Memory is per-client. Outputs are bespoke." | Show vertical template customization |
| "$799/mo is more than a contractor." | "Per-client cost on Workspace tier drops to ~$32/mo if you run 25 clients. Cheaper than a contractor's lunch budget." | Pricing math |
| "We use CrewAI / LangGraph / AutoGen." | "We're not a generic agent framework — we're the supervision + orchestration layer above the execution runtimes. CrewAI is welcome to run inside an OpenClaw-equivalent slot." | Architecture diagram |
| "Compliance — client data isolation." | "Per-client workspace ID, per-client memory namespace, per-client billing ledger. Audited. SOC 2 path active." | Security doc |
| "We resell to clients — branding?" | "White-label is on the Workspace tier roadmap. For pilot, run on Baseline brand; switch on white-label when it ships." | Roadmap |
