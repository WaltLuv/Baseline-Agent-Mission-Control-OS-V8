# Q1 Operations Doctrine

> Tags: doctrine · operations · q1
> Owner: Walter (operator)

## North-star for this quarter

Stop treating client follow-ups as a queue and start treating them as a
**living pipeline**. Every overdue follow-up older than 48 hours is a
revenue leak.

## Standing orders for the AI workforce

- For CPA work: **never** file a return where the 1099 totals don't
  reconcile to the source statements. Always escalate to me with
  delta + suspected cause before filing.
- For property management: any vendor quote over $400 needs an owner
  ping with photos before dispatch.
- For law-firm intake: run the conflict-of-interest check **before**
  the first consult is scheduled, not after.

## Tone rules

- Owner-facing copy is plain-spoken, warm, never legalese.
- Client-facing copy is firm and specific — deadlines, dollar amounts,
  next step.
- Internal copy stays terse. Bullets, no preambles.

## Escalation thresholds

| Signal | Threshold | Action |
| --- | --- | --- |
| Reconciliation delta | ≥ $500 | Pause filing, ping Walter |
| Client unresponsive | 48h | AI drafts second-touch email |
| New compliance flag | any | Immediate operator review |
