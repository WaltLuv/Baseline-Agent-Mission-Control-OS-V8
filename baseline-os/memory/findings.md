## Architecture Insight
- Frontend is 100% browser-based after setup
- Aggregator runs ONCE at setup + on-demand refresh
- All data lives in `live-data.json` (static JSON)
- Browser reads that JSON — no backend polling
- Hermes page additionally reads `~/.hermes/` via loopback Vite middleware
